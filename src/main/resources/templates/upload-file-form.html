Drag & Drop File Upload

Drag & Drop a file here

 

Explanation: It is possible to upload only EXEL format files with a size of no more than 5 MB. The data should be generated on a sheet named "List". The contents of the other sheets will be ignored. The structure of the sheet table should be as follows:

  1. Article,
  2. Size: width
  3. Size: height
  4. Size: depth
  5. Description
  6. Style article
  7. Style name
  8. Manufacturer`s name
  9. Manufacturer`s description
  10. Cost price
  11. Sale price 

Columns: Article, Style article, Manufacturer`s name must be filled in.
